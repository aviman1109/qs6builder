关于 JeedUI

JeedUI 是一套简洁轻便的 Web 控件。它对企业领域常用的功能进行封装，开发者只需要创建控件并按需要调用相关方法，不需要过度关注界面设计和 DOM 操作。从而提高生产效率，快速构建企业级 Web 产品的界面部分。

JeedUI 基于 HTML5，支持 IE10+、Edge 和 Chrome、Firefox、Safari 等浏览器较新的版本，不支持 IE9 或更低版本的 IE 浏览器。

以树控件为例，典型的 JeedUI JavaScript 代码如下：

var tree = Jui.option.Tree.create({
    target: "parent-dom-id",
    multiSelect: true
});
tree.load([
    {id: "1", text: "节点1", children: [
        {id: "1.1", text: "节点1.1"},
        {id: "1.2", text: "节点1.2"}
    ]},
    {id: "2", text: "节点2"},
    {id: "3", text: "节点3"}
]);

JeedUI 分为以下三部分：

  1. 基础控件：可能被多数页面引用的控件，如按钮、各种输入框、工具条、对话框等。在 JeedUI 发行版本中，基础控件被封装在一个 JavaScript 文件中一次性载入。
  2. 可选控件：被部分页面引用的控件，如列表、表单、树、OutlookBar、页签、日历等。缺省情况下，这些控件每个位于一个独立的 JavaScript 文件中，通过 Jui.use("ControlName") 方法载入。
  3. 设计图控件：工作流及其它设计图控件。和可选控件类似，这些控件需要通过 Jui.use("ControlName") 方法载入。

您可以点击左侧菜单中的内容，打开相应的控件示例页面，了解控件的功能。您也可以在打开的页面上按右键，查看框架源代码，了解如何通过 JavaScript 创建控件并对控件进行操作。

为方便查看，本示例系统的多数页面将 HTML、JavaScript、CSS 放在同一 HTML 文件中。在实际开发时，请将 JavaScript 代码放在 *.js 文件中，将 CSS 样式表代码放在 *.css 文件中，以保持代码结构清晰。

本示例系统仅对控件的部分功能予以演示。如需了解全部功能，请参考控件手册。
